-Ofun

It's Christmas 2012. But this ain't about Perl 6. It's about something amazing that can, should, and inevitably will happen to more open source projects:

“Optimizing for fun has had tremendous benefits. In just 8 months, the Pugs project has gained well over 100 committers“
~~ Geoff Broadwell, in an O'Reilly article “-Ofun“ published back in 2005.

“the essence of fun boils down to instant gratification and a sense of wonder and discovery”
~~ Audrey Tang

This website contains:

  • Older material about -Ofun;
  • New variations on -Ofun;
  • A directory of active -Ofun projects;

To get the ball rolling we registered this domain name and created the o-fun github project that stores this content. Feel free to change it as you see fit. We haven't set up anywhere to talk; maybe use the twitter hashtag #Ofun?
